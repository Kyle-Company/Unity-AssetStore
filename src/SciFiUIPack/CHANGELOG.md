# Changelog — Sci-Fi UI Pack

All notable changes to this package are documented here.
This project follows [Semantic Versioning](https://semver.org/).

## [1.0.0] — 2026-06-18

### Added
- 17 nine-sliced UI sprites: buttons (normal / hover / pressed / disabled), panel,
  tooltip, status bar (background + fill), input field, checkbox (on / off), toggle
  (track + knob), tabs (active / inactive), divider, badge.
- 24 monochrome, tintable icons.
- `RainbowBolt/SciFiUI` Canvas shader with optional scanlines, edge glow and holo flicker.
- Runtime components: `HoloButton`, `AnimatedBar`, `HoloPanel`, `ThemeApplier`, `UIThemeSO`.
- Editor tooling: one-click `Build Sci-Fi UI Pack`, automatic 9-slice importer,
  demo-scene generator, `Validate Pack` self-check, and optional Orbitron TMP font generator.
- Demo scene and full documentation.
- Bundled Orbitron font (SIL Open Font License).

### Notes
- Targets Unity 6 (6000.x) and URP. The UI shader is also Built-in compatible.
