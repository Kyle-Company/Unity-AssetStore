## Description (long)

**Ship your game's UI faster.** Sci-Fi UI Pack gives you a complete, consistent set of
holographic interface art and the code to make it move — clean, documented, and built to
drop straight into your project.

No messy slicing, no mismatched styles, no fighting with prefabs. Import, click one menu
item, and you have a themed, animated UI showcase running.

**What you get**
- ✦ **17 nine-sliced sprites** — buttons (normal / hover / pressed / disabled), panel,
  tooltip, status bar, input field, checkbox, toggle, tabs, divider, badge. Resize freely;
  borders are applied automatically.
- ✦ **24 tintable icons** — monochrome and crisp; recolor to any palette in seconds.
- ✦ **Holographic Canvas shader** — optional scanlines, edge glow and flicker. Off by
  default, so it's a safe drop-in.
- ✦ **Drop-in components** — `HoloButton` (hover/press motion + sounds), `AnimatedBar`
  (smooth, gradient health/energy bars), `HoloPanel` (animated open/close).
- ✦ **One-click theming** — a single Theme asset recolors every component and font.
- ✦ **One-click setup** — a menu item builds materials, prefabs and a demo scene for you.
- ✦ **A ready demo scene**, full documentation, and an SDF font (Orbitron, OFL).

**Built for real projects**
- Tidy folders, sensible names, assembly definitions (no full-project recompiles).
- Pixels-perfect, uncompressed UI sprites with correct 9-slice borders.
- TextMeshPro throughout.
- Responsive support from the developer.

**Requirements**
- Unity 2022.3 LTS or newer (Unity 6 recommended)
- Universal Render Pipeline (works in Built-in too)
- TextMeshPro (ships with Unity)
